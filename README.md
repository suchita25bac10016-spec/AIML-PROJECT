Project Overview: 
This project is titled “Student Performance Prediction & Smart Study Recommendation System.”   
The main objective of this project is to predict student academic performance using Machine 
Learning techniques and provide personalized study recommendations. It helps students analyze 
their performance early and improve their learning strategies. 
1. Objectives 
•To predict student performance using Machine Learning 
•To identify factors affecting academic results 
•To provide personalized study recommendations 
•To improve learning efficiency and reduce stress 
2. Dataset Description 
The dataset includes important features such as: 
•Study Hours 
•Attendance 
•Sleep Hours 
•Previous Exam Scores 
•Final Score 
These features are used to train the model and make predictions. 
3.Methodology 
•Data collection and preprocessing 
•Feature selection 
•Model training using Linear Regression 
•Prediction of student performance 
•Generating study recommendations based on results 
4.Technologies Used 
•Python 
•Machine Learning (Linear Regression) 
•Pandas, Scikit-learn 
•Prolog (for recommendation system) 
5. System Working 
User inputs student data → 
Model processes data → 
Predicts final score → 
System gives study recommendations 
6. Applications 
•Schools and colleges 
•Online learning platforms 
•Student counseling systems 
•Academic performance tracking 
7. Future Scope 
•Integration with mobile applications 
•Use of advanced ML algorithms 
•Real-time performance tracking 
•Emotion and stress analysis 
:- Conclusion: This project shows how Machine Learning can help in predicting student performance and improving 
study patterns. It provides useful insights and recommendations that help students achieve better 
academic results.
